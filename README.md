# crawler91
## 自动爬取下载91优秀视频
定时自动爬取91热门视频，自动下载符合的视频。
## 技术栈
- [Spring-Boot](https://github.com/spring-projects/spring-boot)
- [jsoup](https://github.com/jhy/jsoup)
